# md2wechat
# Markdown → 微信公众号排版工具（静态版）
在线使用：https://gokuscraper.cn/md/

这是一个 **已经构建好的静态版本** 的 Markdown 排版工具。

本仓库提供完整的 **静态文件**，无需安装 Node.js、pnpm 或任何开发环境。

只需下载或上传到服务器，即可直接使用。

适合：

- 零基础用户
- 不想配置开发环境的人
- 想快速部署 Markdown 排版工具的人

---


# 使用方法

## 本地使用

1. 下载本仓库
2. 解压文件
3. 打开 `index.html`

即可在浏览器中使用。

---

## 部署到服务器

本项目是纯静态页面，可以部署到任何静态网站服务。

部署步骤：

1. 下载或克隆本仓库
2. 将文件上传到服务器
3. 打开网站地址
4. 开始使用 Markdown 排版公众号文章

---

# 使用流程

1. 在左侧输入 Markdown 内容
2. 右侧实时预览排版效果
3. 点击复制
4. 粘贴到微信公众号后台
